# LIS Form

# Running the app
## Using Docker Compose

1. Start container   
   `docker-compose up -d `   
2. Enter the container   
   `docker exec -it lis_form sh`   
3. Install npm packages   
   `npm install`   
4. Run app at localhost:5173   
   `npm run dev`   

## Using Node
1. Install npm packages   
   `npm install`   
2. Run app at localhost:5173   
   `npm run dev`
